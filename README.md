# Викторина "Столицы мира"

Веб-приложение для изучения столиц мира. Позволяет просматривать список стран, добавлять новые и проверять знания через викторину.

## Возможности

- 📋 Просмотр списка стран и столиц
- ➕ Добавление новых стран с валидацией
- ❓ Интерактивная викторина с автоматической проверкой
- 🗑️ Удаление стран из списка
- 💾 Хранение данных в JSON-файле

## Технологии

- Python 3.14
- Django 6.0
- HTML/CSS
- Pylint (оценка 10/10)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/neonchem/capital_quiz.git

# Перейти в папку проекта
cd capital_quiz

# Установить зависимости
pip install django

# Запустить сервер
python manage.py runserver
```
